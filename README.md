# WPSN SES

AWS SES(Simple Email Service)을 사용하면 편리하게 이메일을 송/수신할 수 있습니다. 가격도 유사 서비스 대비 저렴한 편입니다. AWS 프리 티어 사용자라면 매월 무료로 (EC2로부터) 62,000건을 송신을 수 있습니다.
